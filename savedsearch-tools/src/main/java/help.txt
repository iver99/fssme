The Export option provides way to read the existing searches and  write it to the file in xml format for a given catgegory.


Usage :

    java -jar  UpdateSavedSearch.jar [-export -outputfilepath <file_name_path > -ssfurl <ssf_url> -categoryId <category_id> ]


Arguments :
-export : To retrive the searches 
-outputfilepath :  searches are written to  output file.
-ssf_url : SSF Url    
-servicemanagerurl: Service manager url    
-categoryId   : Category Id 

Note:Please  provide either ssf url  or servicemanagerurl  but not both.




The Import option provide a way to update existing searches and add new searches.

Usage :

    java -jar  UpdateSavedSearch.jar [-import  -inputfilepath <input_file_path> 
    								  -outputfilepath <output_file_path > -ssfurl <ssf_url>  ]



Arguments :
-import : Update existing serach  or Add new serach.
-outputfilepath :  result of import command written to output file.
-ssf_url : SSF Url    
-servicemanagerurl: Service manager url    
-inputfilepath : input file containing searches in xml format.


Note:Please  provide either ssf url  or servicemanagerurl  but not both.
